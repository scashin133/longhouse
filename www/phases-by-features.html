Project Phases By Feature Set

Purpose: To provide a terse summary of the project schedule by listing features with their corresponding use cases and test cases to provide an easy reference for progress on the project.

The type of work (integration/implementation/hybrid) is also listed for each feature; for more information on this means of categorization, consult the feature set.

Work type is catagorized by the work involved in implementing the given feature, either integration of prefabricated components (N), implementation of new components (M), or a hybrid of implementation alongside integration (H).
