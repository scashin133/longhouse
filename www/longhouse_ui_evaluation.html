Scenario:

You are a project manager at a small software development company. The project you are currently managing is called Notepad Xtreme, 'the most xtreme of text editors'. The senior management at your company has chosen to use Longhouse to host its projects and to provide a space for employee collaboration. Your job is to create a project on Longhouse to represent your Notepad Xtreme project and to get it ready for your employees to use.


Task 1: Log into your account.

  • Use the user name longhouse-tester@gmail.com

  • Use the password longhouse-password


Task 2: Create a project to represent Notepad Xtreme.

  • Choose an appropriate Project Name, Summary and Description for this project.

  • For the repository URL, you can simply use: svn://path/to/repository.

  • Add some labels to help describe the project. Enter whatever comes to mind.


Task 3: Edit your project's information.

  • Your boss has just informed you that the marketing team has come up with a slogan for the project.

  • Edit the project's Summary to say:

Notepad Xtreme: Xtremeness is a state of mind.


Task 4: Add more members to your project.

  • Your employees' user names are:

    • ziv@uci.edu

    • patil@uci.edu


Task 5: Assign an issue.

  • You want one of your new members, patil@uci.edu, to upload the Notepad Xtreme documentation to the Downloads section of your project's page.

  • Enter an appropriate Summary and Description for this task.

  • Assign the task to patil@uci.edu

  • Make sure that you're notified when this issue is changed by another user.

  • Make sure that the Labels are appropriate for this task. If you're not satisfied with them, you can change them.


Task 6: Change the issue that you created.

  • Some time has passed and patil@uci.edu has not completed the task.

  • Reassign it to ziv@uci.edu.

  • Make sure that he's notified that the issue has been assigned to him.


Task 7: End your session by signing out.


Task 8: Return to your project page using your existing account.

  • Use the login information in Task 1.

  • Return to the Notepad Xtreme project page.


Task 9: Remove project members.

  • Your employees have left your project for another project.

  • Remove both ziv@uci.edu and patil@uci.edu from the list of project members.


Task 10: Flag your project for deletion.

  • Due to the lack of productivity, management has decided that the Notepad Xtreme project is to be terminated.

  • Flag your project for deletion so that the project is no longer viewable by the public.
